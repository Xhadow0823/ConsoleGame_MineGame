## 操作說明
|操作 |功能    |
|---  |---    |
|方向鍵|移動游標|
|ESC  |退出遊戲|
|F/f  |設為旗幟|
|R/r  |開新遊戲|
|**特殊操作**| |
|S/s  |作弊模式(地雷不爆炸)|
|D/d  |輸掉遊戲|
|S/s  |贏得遊戲|
